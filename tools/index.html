  • Boxen And My Experience

    Boxen is a new tool that allows new Macs to be quickly set-up using the same technologies that sysops have become accustomed to, Puppet. Now Puppet has it’s own language to define what actions are executed on target machine. It was originally developed to help prepare cloud servers for easier management/deployment. The popularity of Puppet has increased and it has become a widely used tool and so the team over at github.com decided to build a system that utilises it to perform similar setup tasks on Macs.
